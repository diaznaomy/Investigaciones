¿Qué es la accesibilidad web?

Es el diseño y desarrollo de sitios web para que todas las personas, incluyendo aquellas con discapacidades, puedan percibir, entender, navegar e interactuar con el contenido.

También considera usuarios con conexiones lentas, dispositivos antiguos o tecnologías de asistencia.

Importancia de la accesibilidad web

  • Igualdad de acceso: Permite el uso universal de Internet.
  • Cumplimiento legal: Exigido por leyes como la Ley 26687 (CR) o ADA (EE.UU.).
  • Mejora UX: Beneficia a todos los usuarios.
  • Mayor alcance: Llega a más personas, incluyendo adultos mayores.
  • SEO y compatibilidad: Mejora el posicionamiento y funcionamiento.

Principios fundamentales (WCAG)

son un conjunto de recomendaciones internacionales para que el contenido web sea accesible para todas las personas,
incluidas aquellas con discapacidades. Estas pautas
se organizan en torno a 4 principios fundamentales:

  1. Perceptible: Contenido visible para todos.
    • Texto alternativo en imágenes.
    • Subtítulos en vídeos.
    • Ejemplo: <img src="grafico.png" alt="Gráfico de barras mostrando el crecimiento de ventas del 2024">
  2. Operable: Navegable con distintos dispositivos.
    • Navegación por teclado.
    • Saltos de navegación (skip links).
  3. Comprensible: Lenguaje y diseño claros.
    • Botones con texto explicativo.
    • Mensajes de error comprensibles.
  4. Robusto: Compatible con distintas tecnologías.
    • Uso de HTML semántico.
    • Validación con herramientas como WAVE.
    • Ejemplo: uso de etiquetas <nav>, <main>, <footer>
